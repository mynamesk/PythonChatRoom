# Chatroom
* the chatroom can accept different clients connection.
* allow clients use the chatroom at same time
* commands of the chatroom
login
say
logout
* easy to extend
## Chatroom command
* login   nickname
* logout 
* say  words
* look
* who
## 如何连接服务器
    * 在命令行里输入 telnet localhost 5005
    5005为为端口号port
